Reference Implementation for Smalltime
======================================

A C implementation to demonstrate smalltime.



Requirements
------------

  * CMake 3.5 or higher
  * A C compiler
  * A C++ compiler (for the unit tests)



C Library Dependencies
----------------------

 * stdint.h: For standard integer types



Building
--------

    mkdir build
    cd build
    cmake ..
    make



Running Tests
-------------

    ./test/smalltime_test
